Count the number of times two chars occur in a string Open on Hyperskill

Write a program that counts the number of times that the two characters occur consecutively in a string. The order of characters should be the same as in the input.

Note, in the string "bbb" , the sequence "bb" occurs two times.

Input data format
The single input line contains a string and two characters.

Sample Input:
abcd b c

Sample Output:
1

Sample Input:
cbbb b b

Sample Output:
2


Memory limit: 256 Mb
Time limit: 5s
