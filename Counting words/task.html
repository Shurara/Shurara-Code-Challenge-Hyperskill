Counting words Open on Hyperskill

Given an array of strings implement these two methods:

  • wordCount which returns a SortedMap<String, Integer> with a key for each different string, with the value the number of times that string appears in the array. The method takes an array of strings as input;
  • printMap which prints all entries of the map ("key : value").

Sample Input:
a b b c d a b
Sample Output:
a : 2
b : 3
c : 1
d : 1


Memory limit: 256 Mb
Time limit: 5s
