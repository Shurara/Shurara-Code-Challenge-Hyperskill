Seconds since start of a day Open on Hyperskill

Write a program that reads a number of seconds from the start of a day and prints the current time.

If the resulting time has zero seconds, do not output them.


Sample Input:
12345
Sample Output:
03:25:45

Sample Input:
60000
Sample Output:
16:40

Sample Input:
3600
Sample Output:
01:00


Memory limit: 256 Mb
Time limit: 5s
