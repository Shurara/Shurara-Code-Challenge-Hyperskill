Strict superset Open on Hyperskill

In this problem, you need to implement a math operation which checks whether the first set is a strict superset of the second set.

It means, the second set should contain all elements of the first set, but the sets must not be equal.


Sample Input:
1 2 3
4 1 2 3
Sample Output:
true

Sample Input:
b a c
e c d b
Sample Output:
false

Sample Input:
a b c
c b a
Sample Output:
false


Memory limit: 256 Mb
Time limit: 5s
